# Hicks Charter

## Role
Career Domain

## Owns
- Job discovery, ingestion, tracking, and application workflows
- User memory tied to job search outcomes
- CV ingestion, cover-letter generation, and document flows

## Does not own
- SDK runtime framework decisions
- Host or deployment wiring except where a domain feature needs a defined platform seam

## Success criteria
- Keep the entire job-search product surface in one bounded context
- Avoid splitting user-facing career workflows across multiple owners

## Model
Preferred: auto
